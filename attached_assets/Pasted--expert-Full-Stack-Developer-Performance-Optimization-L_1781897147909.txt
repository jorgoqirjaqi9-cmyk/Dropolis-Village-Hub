### Εργασίες που πρέπει να κάνεις:

1. **Hero Image Optimization**
   - Βελτίωσε την κύρια εικόνα (stone-bridge-epirus.jpg) → καλύτερο WebP, σωστά sizes & srcset
   - Πρόσθεσε σωστά attributes (width, height, fetchpriority, loading)

2. **Critical CSS & Render Blocking**
   - Inline ή Preload το κρίσιμο CSS
   - Βελτίωσε το loading των fonts (font-display: swap)

3. **JavaScript Optimization**
   - Code splitting / dynamic imports όπου χρειάζεται
   - Defer ή Lazy load μη-κρίσιμα vendor bundles
   - Μείωσε unused JS

4. **Performance Best Practices**
   - Πρόσθεσε Preconnect & Preload hints
   - Βελτίωσε image delivery (sizes, quality)
   - Βελτίωσε το DOM size αν είναι απαραίτητο

5. **Γενικές Βελτιώσεις**
   - Προσθήκη κατάλληλων meta tags για performance
   - Βελτιστοποίηση του viewport
